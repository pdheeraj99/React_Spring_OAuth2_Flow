| Claim | Meaning |
|-------|---------|
| `iss` | Issuer (Google) |

| `email` | User's email |
| `name` | User's display name |
| `picture` | Profile photo URL |
| `iat` | Issued at (timestamp) |
| `exp` | Expires at (timestamp) |
